# BFNT Converter

PC-9801 時代に使われていた BFNT、BFNT+フォーマット（Rev.1.6）のファイルを画像として表示・変換できるソフトウェアです。Windows 11, .NET 6 で動作確認。

[Download](https://github.com/jz5/BfntConverter/releases)
